# Console Logging (QA Perspective)

## Why this matters

Console methods are not just for debugging — they are useful in testing and automation.

### Real QA Use Cases

- `console.error()` → Identify failures in test runs  
- `console.table()` → Validate API response data  
- `console.time()` → Measure performance  
- `console.assert()` → Lightweight assertions during development  

---

## Key Takeaway

Good engineers don’t just write code — they make debugging easy.
